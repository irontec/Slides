## Ubuntu 14.04 and up

Install from repositories:
```bash
$ sudo apt-get update
```
<code class="bash"><strike>$ sudo apt-get install docker</strike></code>


```bash
$ sudo apt-get install docker.io
```

Tab-completion:

```bash
$ source /etc/bash_completion.d/docker.io
```

Avoid using sudo:

```bash
$ sudo adduser $USERNAME docker
```
